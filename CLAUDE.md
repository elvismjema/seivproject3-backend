# Claude Configuration

## IMPORTANT: Git Commit Rules

When making commits:
- NEVER use "Generated by Claude" or similar bylines
- NEVER add "Co-Authored-By: Claude" lines
- NEVER add any Claude-related attributions, references, or emojis
- NEVER mention Claude, AI, or any AI tools in commit messages
- Keep commit messages professional and focused on the changes made

## Example Good Commit Message:
```
Fix Google OAuth login for accounts without last names

- Handle null/undefined family_name from Google OAuth
- Improve error handling and add detailed logging
- Fix session expiration date comparison
```

## Example Bad Commit Message (DO NOT USE):
```
Fix Google OAuth login issues

Generated with Claude Code
Co-Authored-By: Claude <noreply@anthropic.com>
```
